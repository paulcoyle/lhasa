Classpublic class StageLayoutContainer
InheritanceStageLayoutContainer LayoutContainer LayoutElement flash.display.Sprite

A container that manages its own total_width and total_height to match that of the stage. Your application class should extend this class and invoke super() immediately in the constructor. This class defaults to using the FreeLayoutDelegate so that subsequent containers can be placed manually but this is by no means the only way to use this class.
