horizontalproperty 
horizontal:Number  [read-write]

The sum of the right and left values. Setting this property applies the given value to both right and left equally.

Example 

  var box:Box = new Box(1, 2, 3, 4);
  trace(box.right, box.left, box.horizontal);// (2 4 6)
  
  box.horizontal = 10;
  trace(box.right, box.left, box.horizontal);// (10 10 20)
